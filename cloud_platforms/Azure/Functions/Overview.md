## Azure Functions

Azure Functions is a serverless solution that allows you to build robust apps withouth worrying about the underlying infrastructure. You develop your code, deploy it to Azure and that's it.

# Scenarios

Azure Functions cover a wide range of scenarios some of which include:
    - process file uploads;
    - process data in real time;
    - run AI inference;
    - run scheduled tasks;
    - build a scalable web API;
    - build a serverless workflow;
    - respond to database changes;
    - create reliable message systems.

# Development lifecycle

1. You can develop in C#, Java, JavaScript, PowerShell, or Python, or you can use custom handlers for languages like Rust and Go.
2. You can develop and debug locally.
3. After development is finished you can deploy to Azure using CLI, CI/CD pipelines or your IDE.
4. You can monitor performance and diagnose issues with built-in Azure Monitor and Application Insights integration.
